How this tool works

This interface sends your image to the selected model family and returns class probabilities over:

  • Bird-drop
  • Clean
  • Dusty
  • Electrical-damage
  • Physical-Damage
  • Snow-Covered

Model availability
If the selected model is missing, you’ll see a red warning: Model not trained yet and no inference is attempted.

Tips for reliable predictions

  • Use sharp images (no heavy blur / motion).
  • Avoid extreme shadows or strong reflections.
  • Try different backbones once they’re trained and compare results.
